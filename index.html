ИНСТРУКЦИЯ ПО УСТАНОВКЕ

ДЛЯ ANDROID:

1. Нажмите на красную кнопку выше и скачайте файл app-debug.apk.

2. Откройте скачанный файл на телефоне.

3. Если появится окно "Блокировка установки", нажмите "Настройки" и разрешите "Установку из неизвестных источников".

4. Нажмите "Установить" и запустите приложение.

ДЛЯ WINDOWS:

На данный момент прямая версия для Windows находится в разработке. Чтобы использовать Nanklis VPN на ПК сейчас:

1. Скачайте эмулятор Android (например, BlueStacks или NoxPlayer).

2. Перетащите скачанный файл app-debug.apk в окно эмулятора.

3. Запустите VPN внутри эмулятора.
